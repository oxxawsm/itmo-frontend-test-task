# Frontend test-task for ITMO
**Made with React, TypeScript, Next.js, Redux**

## Clone project
```
git clone https://github.com/oxxawsm/itmo-frontend-test-task
```

## Install dependencies
```
npm install
```

## Run project in dev mode
```
npm run dev
```

## Build project
```
npm run build
```
